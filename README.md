# 🌦️ Weather App

This project is a simple and interactive **Weather Web Application** built as part of my **Web Development Internship at Prodigy Infotech**.

## 🚀 Features
- Search weather by city 🔍  
- Displays temperature, condition, and location  
- Clean and animated UI ☁️  
- Real-time data using Weather API  

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- WeatherAPI
